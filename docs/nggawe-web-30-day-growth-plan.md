## 1. Target 30 Hari

### Goal utama
- 30–60 qualified leads
- 10–15 call / konsultasi
- 3–5 closing project
- 1–2 recurring maintenance / SEO retainer

### Target revenue realistis
- **Low:** Rp10–20 juta

- **High:** Rp75 juta+ kalau dapet 1–2 project automation/custom

## 2. Positioning

Jangan jual “jasa website” doang karena terlalu umum.

Positioning utama:

> **Website bisnis yang bukan cuma tampil profesional, tapi juga bisa ditemukan di Google dan otomatis menangkap lead.**

Core angle:
- Website premium
- SEO-ready
- Form / WhatsApp / CRM automation
- Cocok buat bisnis yang mau terlihat serius dan dapet lead lebih rapi

### 3.3 Agency / White-label

Contoh:
- social media agency
- design studio
- marketing freelancer

Pain:
- mereka dapat klien tapi tidak punya team dev kuat
- butuh partner build website cepat
- bisa jadi repeat order

## 4. Offer Utama

Bikin 3 paket biar gampang dijual.

### Paket 1 — Website Starter

**Harga:** Rp3,5–7 juta  
**Target:** UMKM / service kecil.

Isi:
- 1–5 halaman
- responsive
- copywriting basic
- WhatsApp CTA
- SEO basic
- deploy live

CTA:

> **Buat bisnis terlihat lebih profesional dalam 7–10 hari.**

### Paket 2 — Website Growth

**Harga:** Rp8–20 juta  
**Target:** paket utama yang harus didorong.

Isi:
- website 5–10 halaman
- service pages
- blog / SEO structure
- portfolio / testimonial
- form lead capture
- analytics
- WhatsApp tracking
- basic automation Telegram / email notification

CTA:

> **Website + SEO foundation + lead system.**

### Paket 3 — Website + Automation System

**Harga:** Rp20–50 juta+  
**Target:** bisnis yang sudah punya flow dan butuh sistem.

Isi:
- semua Growth
- CRM / simple dashboard
- form → spreadsheet / CRM
- Telegram / WhatsApp notification
- autoresponder email
- follow-up pipeline
- booking / order / inquiry flow
- optional AI automation

CTA:

> **Bukan cuma website, tapi sistem lead handling.**

## Minggu 3 — Outreach + Mini Audit

**Tujuan:** dapet lead aktif.

### Target outreach

Cari 100 prospek:
- 40 service business lokal
- 30 UMKM
- 20 agency / freelancer
- 10 startup / produk kecil

Source:
- Google Maps
- Instagram
- Threads
- LinkedIn
- grup bisnis lokal
- website lama dari pencarian Google

### Daily outreach target
- 15–20 DM / email per hari
- 5 mini audit per hari
- 2–3 follow-up per prospek

### DM Script — Soft Audit

```text
Halo kak, gue dari Nggawe Web.
Tadi gue lihat bisnis kakak, menurut gue ada beberapa peluang biar website/online presence-nya bisa lebih kuat buat dapetin inquiry dari Google dan WhatsApp.
Kalau berkenan, gue bisa kirim audit singkat gratis: tampilan, SEO, CTA, dan peluang automation. Mau gue cekin?
```

### DM Script — Kalau Belum Punya Website

```text
Halo kak, gue lihat bisnis kakak sudah jalan tapi kayaknya belum punya website utama.
Buat bisnis service/local, website bisa bantu calon customer lebih percaya dan lebih gampang nemu dari Google.
Gue bisa bantu bikinin struktur website yang cocok dulu secara gratis. Mau gue kirim contoh mapping-nya?
```

### DM Script — Agency / White-label

```text
Halo kak, gue dari Nggawe Web. Kita bantu build website, SEO-ready landing, dan automation flow.
Kalau kakak ada klien yang butuh website tapi team dev lagi penuh, kita bisa bantu white-label.
Mau gue kirim portfolio dan skema partnernya?
```

### Follow-up H+2

```text
Kak, gue follow up ya. Kalau mau, gue bisa kirim 3 poin audit cepat dulu tanpa perlu call.
```

### Follow-up H+5

```text
Gue sempet mapping singkat: biasanya bisnis kayak kakak butuh website yang fokus ke trust, service page, CTA WhatsApp, dan SEO lokal. Kalau mau, gue bisa bantu breakdown halaman yang dibutuhkan.
```

## Minggu 4 — Closing + Conversion

**Tujuan:** ubah lead jadi project.

### Call structure 30 menit

1. Bisnisnya apa?
2. Target customer siapa?
3. Sekarang lead datang dari mana?
4. Masalah utama:
   - trust?
   - Google?
   - closing?
   - follow-up?
5. Butuh website baru / redesign / SEO / automation?
6. Rekomendasi paket
7. Next step: proposal + DP

### Proposal cepat

Bikin template proposal 1–2 halaman:
- problem
- objective
- scope
- timeline
- deliverables
- price
- payment term
- next step

### Closing offer

Urgency ringan:

> **Slot build bulan ini cuma 3 project biar quality tetap rapi.**

Payment:
- 50% DP
- 30% after design approved
- 20% before launch

Alternatif untuk UMKM:
- 60% DP
- 40% before launch
