## FAQ Section

### Heading

```text
Pertanyaan yang sering ditanyakan.
```

### FAQ 1

Question:

```text
Berapa biaya membuat website di Nggawe Web?
```

Answer:

```text
Biaya mulai dari Rp3,5 juta untuk website starter. Project yang butuh custom design, CMS, SEO, automation, atau fitur khusus akan menyesuaikan scope.
```

### FAQ 2

Question:

```text
Berapa lama proses pembuatan website?
```

Answer:

```text
Website sederhana biasanya 5–10 hari kerja. Website custom dengan SEO dan automation bisa memakan waktu 2–6 minggu tergantung kompleksitas.
```

### FAQ 3

Question:

```text
Apakah website sudah SEO-ready?
```

Answer:

```text
Ya. Struktur dasar seperti metadata, sitemap, robots.txt, heading, speed, schema basic, dan Search Console bisa disiapkan sejak awal.
```

### FAQ 4

Question:

```text
Apakah bisa dibantu automation juga?
```

Answer:

```text
Bisa. Form website bisa dihubungkan ke database, spreadsheet, email, Telegram, WhatsApp notification, atau workflow n8n sesuai kebutuhan.
```

### FAQ 5

Question:

```text
Apakah menerima project white-label dari agency?
```

Answer:

```text
Ya. Nggawe Web bisa membantu agency sebagai partner build untuk website, landing page, technical SEO, dan automation secara white-label.
```
