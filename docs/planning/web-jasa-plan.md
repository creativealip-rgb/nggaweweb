## 8. Tech Stack Rekomendasi

### Stack Utama

- Frontend: Next.js
- Styling: Tailwind CSS
- Animation: Framer Motion
- CMS: Sanity / Payload
- Form: React Hook Form + webhook
- Automation: n8n
- Hosting: Vercel
- Analytics: Plausible + GA4
- SEO: sitemap, robots, schema JSON-LD
- Email: Resend
- Database jika perlu: Supabase

### Alasan

- Cepat
- SEO kuat
- UI premium
- Mudah integrasi automation
- Cocok buat portfolio teknis
- Mudah scale ke fitur tools dan client portal
