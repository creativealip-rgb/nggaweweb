# Master Plan — Nggawe Web

## 0. Keputusan Final

Nama brand kerja: **Nggawe Web**

Fokus utama:

- Website premium
- SEO lokal dulu
- Business automation
- Subcontract untuk agency
- Startup/UMKM service business

Positioning:

> Nggawe Web bikin website cepat, SEO-ready, dan sistem automation buat bisnis yang mau kelihatan profesional, gampang ditemukan, dan lebih rapi ngurus lead.

Tagline opsi utama:

> Website yang tampil bagus, ditemukan Google, dan bantu bisnis jalan lebih otomatis.

Style acuan:

- Knapsack Creative: jelas, service-business oriented, SEO + conversion kuat
- Pocketknife: clean, premium, one-stop digital partner, human tone
- Nggawe Web: versi lebih tech-forward karena ada automation

## 3. Offer & Packages

Pricing boleh diubah nanti. Ini baseline agar copy dan funnel jelas.

### Package 1 — Website Starter

Untuk UMKM/service business yang butuh website profesional cepat.

Harga awal: **mulai Rp3.500.000**

Deliverables:

- 1–5 halaman
- Responsive design
- Basic SEO setup
- WhatsApp CTA
- Contact form
- Google Analytics/Search Console setup
- Basic speed optimization

Timeline: 5–10 hari kerja

### Package 2 — Website Growth

Untuk bisnis yang mau website lebih serius dengan SEO foundation.

Harga awal: **mulai Rp8.500.000**

Deliverables:

- 5–10 halaman
- Custom design
- CMS blog/portfolio
- SEO structure
- Schema basic
- Lead form
- WhatsApp tracking
- Performance optimization
- Basic automation: form → email/Google Sheet/Telegram

Timeline: 2–4 minggu

### Package 3 — Website + SEO System

Untuk service business yang mau grow lewat organic search.

Harga awal: **mulai Rp15.000.000**

Deliverables:

- Website custom
- Keyword map
- Service pages SEO
- Blog setup
- Local SEO setup
- Technical SEO
- Schema expanded
- Internal linking
- 5 artikel awal
- Monthly SEO report template

Timeline: 4–6 minggu

### Package 4 — Automation System

Untuk bisnis yang mau hemat waktu dan rapikan proses.

Harga awal: **mulai Rp5.000.000**

Deliverables:

- Workflow audit
- n8n/Make automation
- Form webhook
- CRM/spreadsheet sync
- Auto notification
- Email/WhatsApp follow-up setup
- Documentation

Timeline: 1–3 minggu

### Package 5 — Agency White-Label

Untuk agency yang butuh partner build.

Harga: **custom / retainer**

Deliverables:

- Landing page build
- Full website build
- CMS setup
- Technical SEO
- Bug fix/maintenance
- Automation integration
- NDA/white-label support
